The Basic Lenguage of the Web: HTML


Posted by Natale Jones on Monday, June 23 2023



All modern websites and web applications are built using three fundamental technologies: HTML CSS and JavaScript. These are the lenguages of that we will learn in Riwi.
In this post, let`s focus on HTML. We will learn what HTML is all about, and why you too should learn it


What is HTML?

HTML stands for Hyper Text Markup Lenguage. It`s a markup lenguage that use to structure and describe the content of a webpage(not a programming lenguage).
HTML consit of elements that describe diferent types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn mor at MDN Web Docs.


Why should you learn HTML?

There are countless reasons for learning the fundamental lenguage of the web. Here are 5 of them:

  • To be able to use the fundamental web dev lenguage
  • To hand-craft beautiful websites instead of relying on tools like Worpress or Wix
  • To build web applications
  • To impress friends
  • To have fun -emoji-
